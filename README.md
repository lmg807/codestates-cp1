1. 데이터셋 설명

    #### commit.xlsx (커뮤니티 댓글 데이터)
    - no. : 번호
    - site : 커뮤니티 사이트
    - writer : 글쓴이
    - comment : 댓글내용
    - sympathy : 공감수

    #### news.xlsx (뉴스기사 데이터)
    - no : 번호
    - datetime : 기사입력일
    - press : 언론사
    - writer : 기자
    - title : 기사제목
    - article : 기사내용
